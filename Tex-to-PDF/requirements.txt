# Rendering Engine
MiKTeX is required for this project's default Windows setup.
It provides the actual PDF rendering engines: xelatex and lualatex.
Download MiKTeX here: https://miktex.org/download

# macOS/Linux users can also use their OS-native TeX distribution, such as
# MacTeX on macOS or TeX Live on Linux, as long as xelatex or lualatex is
# available on PATH. MiKTeX is also an option.
