A personal collection of reusable skills, workflows, and guidance for AI agents.

  This repository is a central place to store agent knowledge that can be reused across
  projects. It is intended for coding agents and other AI agents that need clear, practical
  instructions for specific tasks or areas of expertise. It will continue to be updated
  over time.

  ## Structure

  Each top-level folder represents a separate skill, workflow, capability, or subject area.

  Folders are organized by purpose. A skill folder may include instructions, examples,
  references, templates, or helper files needed to use or maintain that skill.

  Each skill should be understandable on its own, while still being easy to combine with
  other skills when a task needs more than one capability.
